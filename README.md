# Planning Poker (Mattermost)

Бот для канала планирования: корневой пост с ссылкой на Jira (`/browse/KEY-123`) и `@mentions` запускает раунд; бот **сам пишет в ЛС** каждому упомянутому с напоминанием и ссылкой на тикет; голос — ответом числом в этом ЛС; итог и медиана (при чётном числе оценок — среднее двух центральных, округление вверх) публикуется в тред.

## Настройка

1. Скопируйте `.env.example` в `.env`, укажите `MATTERMOST_URL`, `BOT_TOKEN`, при необходимости `PLANNING_CHANNEL` и `MATTERMOST_PORT`.
2. Добавьте бота в команду и в канал (например `dm-planning-poker`).
3. Установите зависимости: `pip install -r requirements.txt`

## Запуск

Из корня репозитория:

```bash
python -m src.main
```

Нужен доступ к серверу Mattermost (VPN, если требуется).

### TLS / сертификат

Если видите `CERTIFICATE_VERIFY_FAILED`, у внутреннего сервера часто свой CA. Варианты: задать в `.env` `MATTERMOST_SSL_CA_FILE=/путь/к/ca.pem` или временно `MATTERMOST_SSL_VERIFY=false` (только в доверенной сети).

## Права

Для постов в канал и треды — как минимум возможность писать в канал. Для приёма оценок в ЛС пользователи пишут боту первыми; если события из ЛС не приходят, проверьте настройки бота и режим приватности.

## Параллельные раунды

Один и тот же пользователь не может участвовать в двух незавершённых раундах: второй старт в канале будет отклонён с сообщением в тред.
