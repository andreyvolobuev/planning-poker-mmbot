# Planning Poker (Mattermost)

Бот реагирует на **корневые** посты в **любых командных каналах**, куда он добавлен: в тексте должны быть ссылка на Jira (`/browse/KEY-123`) и `@mentions` участников. В ЛС каждому уходит приглашение со **ссылкой на тред** в канале (`…/team/pl/post_id`) и инструкцией. Оценка — **ответом в треде на это приглашение** (одно число, точка или запятая как разделитель; шаг **0.05**, лишняя точность округляется вверх, напр. `2.93` → `2.95`). До итога можно обновить числом в том же треде. В исходном треде публикуются результаты: **медиана** (как раньше, с шагом 0.05 при чётном числе голосов) и **среднее арифметическое** по учтённым оценкам.

`MATTERMOST_URL` в `.env` должен совпадать с тем, как вы открываете Mattermost в браузере (от этого собираются permalinks в ЛС).

## Настройка

1. Скопируйте `.env.example` в `.env`, укажите `MATTERMOST_URL`, `BOT_TOKEN`, при необходимости `MATTERMOST_PORT` и параметры TLS.
2. Добавьте бота в нужные **командные** каналы (открытые, закрытые или групповые — типы O, P, G).
3. Установите зависимости: `pip install -r requirements.txt`

## Запуск

Из корня репозитория:

```bash
python -m src.main
```

Нужен доступ к серверу Mattermost (VPN, если требуется).

### TLS / сертификат

Если видите `CERTIFICATE_VERIFY_FAILED`, у внутреннего сервера часто свой CA. Варианты: задать в `.env` `MATTERMOST_SSL_CA_FILE=/путь/к/ca.pem` или временно `MATTERMOST_SSL_VERIFY=false` (только в доверенной сети).

## Права

Для постов в канал и треды — как минимум возможность писать в канал. Для ЛС бот сам открывает диалог; если доставка не уходит, проверьте приватность сообщений.

## Несколько задач сразу

Один и те же люди могут параллельно участвовать в нескольких раундах: в ЛС у каждой задачи своё приглашение и свой тред, оценка однозначно привязана к `root_id` этого треда.

### Досрочное завершение (`/finish`)

Если кто-то из оценивающих не отвечает, **автор поста**, с которого запущено голосование, может написать в **этом же треде в канале** команду `/finish`. Тогда бот подводит итог **только по уже пришедшим** голосам (медиана по ним), перечисляет не проголосовавших и закрывает раунд.
